# RAG Chatbot for Physical AI & Humanoid Robotics

## Introduction

Welcome to the RAG (Retrieval-Augmented Generation) chatbot for the Physical AI & Humanoid Robotics textbook. This AI-powered assistant can answer questions specifically from the textbook content using the Gemini 1.5 Flash API. Ask any questions about humanoid robots, ROS 2, digital twins, NVIDIA Isaac platform, vision-language-action models, hardware, or the capstone project.

## How It Works

1. **Query**: You ask a question about the textbook content  
2. **Retrieval**: The system retrieves relevant textbook passages  
3. **Generation**: Gemini 1.5 Flash generates an answer based on the retrieved content  
4. **Response**: You receive a contextual answer with source references
